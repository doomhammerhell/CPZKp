# Contributing

This chapter covers how to contribute to CPZKp.

## Development Setup

1. Install Rust:
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

2. Clone repository:
```bash
git clone https://github.com/yourusername/cpzkp.git
cd cpzkp
```

3. Install dependencies:
```bash
cargo build
```
